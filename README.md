## Prerequisites<!--#prerequisites-->
Ubuntu server 23.10

## How to start<!--#how-to-start-->
- sudo apt update
- sudo apt upgrade
- sudo apt install ansible
- cd /home/user/.ansible
- touch hosts
- touch ansible.cfg

## Local computer<!--#local-computer-->
- cd /tmp
- git clone https://github.com/div-cz/div_ansible.git
